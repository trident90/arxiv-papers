## 🤖 Artificial Intelligence (cs.AI) - 5개

### 1. **OS-Themis: GUI 에이전트를 위한 확장 가능한 보상 프레임워크**
- **제목**: A Scalable Critic Framework for Generalist GUI Rewards
- **저자**: Zehao Li, Zhenyu Wu, Yibo Zhao et al.
- **URL**: https://arxiv.org/abs/2603.19191
- **요약**: GUI 에이전트의 강화학습 훈련을 위한 OS-Themis 프레임워크를 제시하며, 다단계 검증 메커니즘으로 보상 신뢰성을 향상시킵니다. AndroidWorld 벤치마크에서 온라인 RL 훈련 시 10.3% 성능 개선을 달성했으며, 자가 학습 루프에서 6.9% 추가 이득을 보고합니다. 실제 GUI 자동화 시스템 구축에 직접 적용 가능한 기술입니다.

### 2. **Box Maze: LLM의 신뢰성 높은 추론을 위한 프로세스 제어 아키텍처**
- **제목**: A Process-Control Architecture for Reliable LLM Reasoning
- **저자**: Qiang Zou et al.
- **URL**: https://arxiv.org/abs/2603.19182
- **요약**: LLM의 기존 출력 필터링 방식이 아닌 명시적 프로세스 제어 계층(메모리 그라운딩, 구조화된 추론, 경계 강화)을 통해 할루시네이션을 방지합니다. 적대적 공격 시 경계 실패율을 40%에서 1% 이하로 감소시키며, 프로덕션 LLM 시스템의 신뢰성 개선에 즉시 활용 가능합니다.

### 3. **cuGenOpt: GPU 가속 범용 메타휴리스틱 조합 최적화 프레임워크**
- **제목**: A GPU-Accelerated General-Purpose Metaheuristic Framework for Combinatorial Optimization
- **저자**: Yuyang Liu et al.
- **URL**: https://arxiv.org/abs/2603.19163
- **요약**: TSP, VRP 등 조합 최적화 문제를 CUDA로 병렬화한 프레임워크로, 기존 MIP 솔버 대비 자릿수 성능 향상을 달성합니다. Python API와 LLM 기반 문제 설명 변환기를 제공하며, 물류/스케줄링 최적화에 실제 배포 가능한 엔진입니다.

### 4. **D5P4: 이산 확산 텍스트 생성에서 다양성 보장**
- **제목**: Partition Determinantal Point Process for Diversity in Parallel Discrete Diffusion Decoding
- **저자**: Jonathan Lys et al.
- **URL**: https://arxiv.org/abs/2603.19146
- **요약**: 이산 확산 모델의 병렬 디코딩에서 빔 서치를 일반화하고 행렬식 점 과정으로 다양성을 통제합니다. 생성 품질을 유지하면서 다양성을 개선하며, 다중 GPU 환경에 최적화되어 대규모 텍스트 생성 시스템에 바로 적용 가능합니다.

### 5. **LLM 기반 바이너리 분석의 암묵적 패턴 분석**
- **제목**: Implicit Patterns in LLM-Based Binary Analysis
- **저자**: Qiang Li et al.
- **URL**: https://arxiv.org/abs/2603.19138
- **요약**: 521개 바이너리의 99,563 추론 단계 분석을 통해 LLM이 암묵적으로 형성하는 패턴 4가지(조기 제거, 경로 종속 고착, 목표 백트래킹, 지식 가이드 우선순위)를 규명합니다. 보안 분석 시스템 설계와 LLM 에이전트 신뢰성 향상에 필수적인 인사이트입니다.

## 🧠 Machine Learning (cs.LG) - 5개

### 1. **온라인 학습과 균형 계산: 랭킹 피드백 활용**
- **제목**: Online Learning and Equilibrium Computation with Ranking Feedback
- **저자**: Mingyang Liu et al.
- **URL**: https://arxiv.org/abs/2603.19221
- **요약**: 정확한 유틸리티 값 대신 순위 정보만으로 온라인 학습이 가능함을 증명하고 부분 관측성 환경에서도 준선형 후회 경계를 달성합니다. LLM 라우팅 및 게임 이론 응용 분야에서 프라이버시를 보존하면서 강력한 성능을 제공합니다.

### 2. **피싱 탐지: 특징 경제학이 지배하는 적대적 강건성**
- **제목**: Robustness, Cost, and Attack-Surface Concentration in Phishing Detection**
- **저자**: Julian Allagan et al.
- **URL**: https://arxiv.org/abs/2603.19204
- **요약**: UCI 피싱 벤치마크에서 모든 모델이 비용 기반 우회 공격에 같은 방식으로 취약함을 보이며, 모델 복잡도보다 특징 경제학이 강건성을 결정합니다. 보안 시스템 설계 시 특징 표현 자체 개선이 필수적임을 증명하는 실무 지침입니다.

### 3. **CALM: 공변량 불일치 하에서 RCT 기반 처치 효과 추정**
- **제목**: Improving RCT-Based Treatment Effect Estimation Under Covariate Mismatch via Calibrated Alignment
- **저자**: Amir Asiaee et al.
- **URL**: https://arxiv.org/abs/2603.19186
- **요약**: 임상 시험(RCT) 데이터와 관찰 데이터의 특징이 다를 때 임베딩 정렬과 보정을 통해 조건부 평균 처치 효과를 정확히 추정합니다. 51개 시뮬레이션 설정 중 22개 비선형 영역에서 기존 방법 대비 우월한 성능을 보여 인과 추론 실무에 가치 있습니다.

### 4. **MIDST: 확산 모델 기반 합성 표 데이터의 프라이버시 평가**
- **제목**: MIDST Challenge at SaTML 2025: Membership Inference over Diffusion-models-based Synthetic Tabular data
- **저자**: Masoumeh Shafieinejad et al.
- **URL**: https://arxiv.org/abs/2603.19185
- **요약**: 확산 모델의 합성 표 데이터가 멤버십 추론 공격에 얼마나 강한지 정량평가하는 벤치마크를 제시합니다. 다중 테이블 제약 조건을 포함한 복합 표 구조에 대한 공격법을 개발하여 프라이버시 보존 데이터 공개 시스템 검증에 필수적입니다.

### 5. **SOL-ExecBench: 하드웨어 한계에 대한 GPU 커널 벤치마킹**
- **제목**: Speed-of-Light Benchmarking for Real-World GPU Kernels Against Hardware Limits
- **저자**: Edward Lin, Sahil Modi, et al. (NVIDIA 팀)
- **URL**: https://arxiv.org/abs/2603.19173
- **요약**: 124개 프로덕션 AI 모델에서 추출한 235개 CUDA 커널 최적화 문제로 구성된 벤치마크입니다. 소프트웨어 베이스라인 대비 성능이 아닌 하드웨어 Speed-of-Light 경계에 대한 성능으로 평가하여, AI 에이전트의 커널 최적화 능력을 객관적으로 측정합니다.

## 👁️ Computer Vision (cs.CV) - 5개

### 1. **VEGA-3D: 비디오 생성 모델의 암묵적 3D 프라이어 활용**
- **제목**: Unleashing Implicit 3D Priors for Scene Understanding
- **저자**: Xianjin Wu et al.
- **URL**: https://arxiv.org/abs/2603.19235
- **요약**: 사전 학습된 비디오 확산 모델에 내재된 3D 구조 프라이어를 추출하여 다중모달 LLM에 기하학적 이해를 부여합니다. 명시적 3D 감독 없이 3D 장면 이해, 공간 추론, 실제 조작 벤치마크에서 우수한 성능을 달성하여 3D 데이터 부족 문제를 우아하게 해결합니다.

### 2. **Matryoshka GS: 연속 레벨-오브-디테일 3D 렌더링**
- **제목**: Matryoshka Gaussian Splatting
- **저자**: Zhilin Guo, Boqiao Zhang et al.
- **URL**: https://arxiv.org/abs/2603.19234
- **요약**: 단일 3D Gaussian Splatting 모델에서 성능 저하 없이 연속적인 속도-품질 트레이드오프를 가능하게 하는 MGS 훈련 프레임워크입니다. 임의의 스플랫 예산에 대해 일관된 품질 향상을 제공하여 모바일 기기부터 고성능 시스템까지 다양한 하드웨어에 단일 모델 배포 가능합니다.

### 3. **CubiD: 고차원 표현 토큰으로 이산 시각 생성**
- **제목**: Discrete Visual Generation on High-Dimensional Representation Tokens
- **저자**: Yuqing Wang et al.
- **URL**: https://arxiv.org/abs/2603.19232
- **요약**: 기존 8-32차원 토큰의 한계를 벗어나 768-1024차원 표현에 대한 첫 이산 확산 생성 모델입니다. 세밀한 마스킹으로 고차원 상관관계를 학습하며 동일 토큰으로 이해와 생성 모두 가능하게 하여 통합 다중모달 아키텍처의 길을 엽니다. CVPR 2026 채택.

### 4. **MonoArt: 단일 이미지로부터 관절 3D 객체 재구성**
- **제목**: Progressive Structural Reasoning for Monocular Articulated 3D Reconstruction
- **저자**: Haitian Li et al.
- **URL**: https://arxiv.org/abs/2603.19231
- **요약**: 단일 이미지로부터 기하학, 부품 구조, 동작 매개변수를 동시 추정하되, 점진적 구조적 추론으로 모션 큐와 구조의 얽힘 문제를 해결합니다. 다중 뷰 감독 없이 PartNet-Mobility에서 최고 성능을 달성하며 로봇 조작에 직접 적용 가능합니다.

### 5. **SAMA: 명령 기반 비디오 편집의 의미론과 동작 분해**
- **제목**: Factorized Semantic Anchoring and Motion Alignment for Instruction-Guided Video Editing
- **저자**: Xinyao Zhang, Wenkai Dong et al.
- **URL**: https://arxiv.org/abs/2603.19228
- **요약**: 비디오 편집을 의미론 고정(스파스 프레임의 의미 토큰 + 비디오 레이턴트)과 동작 정렬로 분해하여, 외부 VLM 프라이어 없이 순수 학습 기반으로 정밀한 의미 수정과 자연스러운 동작 보존을 동시 달성합니다. Kling-Omni 같은 상용 시스템과 경쟁 가능한 오픈 소스 솔루션입니다.

## 🌐 Networking (cs.NI) - 5개

### 1. **통합 에너지 성능 관리: 5G/6G 네트워크 에너지 절감 기술**
- **제목**: Holistic Energy Performance Management: Enablers, Capabilities, and Features
- **저자**: Meysam Masoudi et al.
- **URL**: https://arxiv.org/abs/2603.18841
- **요약**: 5G NR 네트워크의 다양한 절전 기능들을 단독 활성화했을 때 최적 아닌 결과가 나오는 문제를 해결하기 위해 기능 오케스트레이터 개념을 제시합니다. Lean NR, 스케줄링, 고급 슬립 모드를 조정하여 처리량 손실 최소화(무시할 수준)로 gNodeB 에너지 소비를 대폭 감소시키며 6G 표준화에 기여합니다.

### 2. **iParts: 의도 보존 및 위험 제어 모바일 크라우드센싱**
- **제목**: Risk-Aware Potential Game-empowered Two-Stage Mobile Crowdsensing
- **저자**: Houyi Qi et al.
- **URL**: https://arxiv.org/abs/2603.18670
- **요약**: 개인화된 로컬 차등프라이버시로 작업자의 의도/선호 정보를 보호하면서 질 모델을 구축하고, 위험 인식 사전 계획 및 온라인 임시 모집을 포텐셜 게임으로 모델링합니다. 프라이버시-유틸리티-효율 트레이드오프 최적화로 작업 완료율 향상과 중복 감소를 동시 달성합니다.

### 3. **AutORAN: LLM 기반 O-RAN xApp 자동 개발**
- **제목**: LLM-driven Natural Language Programming for Agile xApp Development
- **저자**: Xin Li et al.
- **URL**: https://arxiv.org/abs/2603.18604
- **요약**: 사용자의 고수준 의도를 분 단위에 배포 가능한 xApp으로 변환하는 LLM 기반 자동화 프레임워크입니다. 수동 개발에 수개월 소요되던 xApp 개발 사이클을 수분으로 단축하여 O-RAN 혁신 롤아웃을 획기적으로 가속화합니다.

### 4. **Wi-Fi 7 MLO의 교차 계층 트래픽 할당과 LSTM-SAC 최적화**
- **제목**: Cross-Layer Traffic Allocation and Contention Window Optimization for Wi-Fi 7 MLO
- **저자**: Zhang Liu et al.
- **URL**: https://arxiv.org/abs/2603.18602
- **요약**: Wi-Fi 7의 다중 링크 동작(MLO)에서 상위 MAC 계층의 트래픽 할당과 하위 MAC 계층의 경합 윈도우를 LSTM 기반 소프트 액터-크리틱으로 동시 최적화합니다. 부분 관측성 네트워크 환경에 적응하여 기존 솔루션 대비 현저한 성능 향상을 달성합니다.

### 5. **RUBICONe: 분산 합의 기반 차선 변경 의사결정**
- **제목**: Wireless RAFT-Unified Behaviors for Intervehicular Cooperative Operations
- **저자**: Zhenghua Hu et al.
- **URL**: https://arxiv.org/abs/2603.18595
- **요약**: RAFT 합의 알고리즘을 무선 통신으로 확장하여 다수 자동주행차량이 지각 정보를 공유·통합한 집단 의사결정으로 차선 변경 판단의 신뢰성을 대폭 향상시킵니다. 소프트웨어 정의 라디오 실험으로 검증되어 V2X 통신 시스템 설계에 직접 적용 가능합니다.
