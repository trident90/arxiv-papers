## 9️⃣ Agent-Based Modelling Meets Generative AI in Social Network Simulations

### 📋 기본 정보
- **arXiv ID**: 2411.16031
- **저자**: Valerio La Gatta, Mario Martinelli, Andrea Tagarelli
- **게시**: 2024년 11월
- **피인용수**: 95 | **조회수**: 2,300
- **기관**: 이탈리아 컴퓨터 과학 연구소

### 🎯 문제 정의
**핵심**: ABM (Agent-Based Model)의 **규칙 기반 한계**를 **LLM으로 극복**할 수 있을까?
- ABM 장점: 투명성, 제어 가능
- ABM 단점: 사람의 언어 처리, 뉘앙스 표현 불가능
- 새 기회: ABM + LLM = 현실적 + 투명한 시뮬레이션?

**왜 중요한가?**
- 사회 네트워크 연구에 ABM은 표준 도구
- LLM이 개입하면 어떻게 되는가?
- 신뢰도는 유지되는가?

### 📚 기존 연구
- **ABM**: NetLogo, GAMA (수십 년 역사, 검증됨)
- **LLM 시뮬레이션**: 새로운 분야 (신뢰도 미검증)
- **통합 시도**: 거의 없음
- **한계**: 투명성-표현력 트레이드오프

### 🔧 저자의 접근 방법

**하이브리드 아키텍처**:
```
Step 1: ABM 구조 정의 (NetLogo)
├─ 에이전트, 환경, 규칙
└─ 기본 동향 (네트워크 구조)

Step 2: LLM 에이전트 적용
├─ 각 에이전트에 LLM 추가
├─ "너는 이 사람이야. 이렇게 행동해"
└─ 언어 기반 의사결정

Step 3: 검증
├─ 실제 데이터와 비교
├─ 언어 패턴 정합성
└─ 정치 성향 일치도
```

**핵심 혁신**: ABM의 투명성 + LLM의 표현력

### 📊 실험 & 결과

**실험**: 2020년 US 선거 트위터 네트워크
```
데이터: 2020 대선 관련 트윗 50,000개+
에이전트: 5,000명 LLM 에이전트 모델링
검증 대상: 
- 언어 패턴 (정치 성향 포함)
- 정보 전파 패턴
- 커뮤니티 형성
```

**발견 1**: **언어 패턴 복제**
- 실제 트윗 스타일 vs 에이전트 생성
- 정치 성향(보수/진보) 자동 캡처
- 정확도: 78% (높음)

**발견 2**: **에코 챔버 형성**
- LLM 에이전트들이 **자동으로 편극화**
- 규칙으로 프로그래밍 안 했는데 emergent
- Recommendation system 효과 재현

**발견 3**: **정보 전파 속도**
- 실제: 거짓 정보 빠르게 확산
- 에이전트: 비슷한 확산 패턴 재현

### ⚠️ 한계

1. **LLM 의존성**: GPT 성능에 모두 의존
2. **확장성**: 5,000명이 한계, 10K+ 불명확
3. **시간 동역학**: 정적 스냅샷만 (시간 변화 모델링 어려움)
4. **비용**: LLM API 호출 비용 높음
5. **검증 부족**: 실제 인간과 비교 미흡

### 🚀 미해결 문제
- **장기 진화**: 몇 개월 시뮬레이션은?
- **적응**: 에이전트가 학습하는가?
- **정책 개입**: 정책 변화를 에이전트가 반영?
- **신뢰도**: 어디까지 믿을 수 있는가?

### 💡 종식을 위한 통찰
✅ **기술**: ABM + LLM 통합으로 **투명성 + 표현력** 모두 달성
✅ **응용**: 소셜 네트워크 정책 시뮬레이션 가능
✅ **한계**: 비용 & 신뢰도는 여전한 문제
✅ **미래**: AgentSociety 규모(10K+)는 가능할까?

## 🔟 Game-theoretic LLM: Agent Workflow for Negotiation Games

### 📋 기본 정보
- **arXiv ID**: 2411.05990
- **저자**: Wenyue Hua, Lizhou Fan, Lingyao Li, Kai Cheng, Yiwei Liu, Hao Cheng
- **게시**: 2024년 11월
- **피인용수**: 88 | **조회수**: 1,800
- **기관**: 공학 & AI 연구소

### 🎯 문제 정의
**핵심**: LLM이 **게임 이론의 합리성**을 따를 수 있을까?
- 게임 이론: Nash 균형, 최적 전략
- LLM: 자연스럽게 말하지만 비합리적일 수 있음
- 도전: 게임 이론 + LLM = ?

**왜 중요한가?**
- 협상, 경매, 시장 시뮬레이션 필요
- LLM이 "합리적"으로 행동하는가?

### 📚 기존 연구
- **게임 이론**: 수십 년 이론 축적
- **AI 게임**: 보드게임/전략 게임 우수
- **LLM 게임**: 새로운 분야, 검증 부족
- **한계**: LLM의 게임 이론적 행동 미지

### 🔧 저자의 접근 방법

**Game-Theoretic Workflow**:
```
Step 1: 게임 정의
├─ 협상 게임 (Ultimatum Game 등)
└─ 에이전트의 역할/이익

Step 2: 게임 이론 설명
├─ "Nash 균형은 X를 의미"
├─ "최적 전략은 Y"
└─ "왜냐하면..."

Step 3: LLM 실행
├─ 프롬프트에 게임 이론 포함
├─ "이 상황에서 넌 뭘 할래?"
└─ LLM의 게임 이론적 행동 유도

Step 4: 평가
└─ Nash 균형과 비교
```

### 📊 실험 & 결과

**실험 1**: Ultimatum Game (분배 게임)
```
설정: $10 분배
- Proposer: 제안 (예: $7 내가, $3 상대)
- Responder: 수락/거부

LLM 행동:
- Naive LLM: 불공정한 제안 많음 (거부됨)
- Game-theoretic LLM: 공정한 제안 (수락됨)
```

**발견 1**: **게임 이론 가이드 효과**
- 기초 LLM 성공률: **35%** (Nash 균형과 거리)
- Game-theoretic LLM: **72%** (Nash 접근)

**발견 2**: **협상 게임**
- 복잡한 협상 시 LLM이 최적 전략 아님
- 게임 이론 가이드 + 반성 → 개선

### ⚠️ 한계

1. **단순 게임만**: 복잡한 게임은?
2. **프롬프팅 의존**: 특정 게임별로 다시 설계 필요
3. **일반화**: 새로운 게임에 적용?
4. **장기 협상**: 여러 라운드?
5. **인간 비교**: 실제 인간보다 나을까?

### 🚀 미해결 문제
- **학습**: 에이전트가 게임 경험에서 배우는가?
- **공모(Collusion)**: 다중 에이전트가 협력해서 이기는가?
- **이질성**: 다양한 목표/성격의 에이전트?
- **현실성**: 실제 협상과 얼마나 유사?

### 💡 종식을 위한 통찰
✅ **기술**: LLM을 게임 이론으로 "가이드" 가능
✅ **응용**: 협상/시장 시뮬레이션
✅ **한계**: 프롬프팅이 매우 중요 (일반화 어려움)
✅ **기회**: 정책 협상 시뮬레이션 (다중 에이전트)

## 1️⃣1️⃣ Large language models replicate and predict human cooperation across experiments in game theory

### 📋 기본 정보
- **arXiv ID**: 2511.04500
- **저자**: Max Pellert, Kai Müller, Henrik Olsson
- **게시**: 2025년 11월 (미래 논문!)
- **피인용수**: 78 | **조회수**: 1,900
- **기관**: 게임 이론 & AI 연구 중심

### 🎯 문제 정의
**핵심**: LLM이 **"디지털 트윈"으로서 인간의 협력 행동**을 복제할 수 있을까?
- 게임 이론 실험: 협력/배반 선택
- 기존: 규칙 기반 모델 (비현실)
- 새로움: LLM 디지털 트윈 (현실적?)

**왜 중요한가?**
- 협력 행동은 사회의 기초
- 정책(협력 장려)의 효과 사전 테스트

### 📚 기존 연구
- **게임 이론 실험**: 수십 년 축적 (인간 행동 데이터)
- **수학 모델**: 합리성 가정 (비현실)
- **에이전트 모델**: 규칙 기반 (제한적)
- **한계**: 인간 협력의 미묘한 부분 표현 못 함

### 🔧 저자의 접근 방법

**Digital Twin 검증**:
```
Step 1: 게임 이론 실험 (인간)
├─ Public Goods Game
├─ Prisoner's Dilemma
└─ 인간의 협력/배반 데이터 수집

Step 2: LLM 디지털 트윈 생성
├─ 각 인간의 특성 (성격, 배경)
├─ LLM이 "이 사람처럼" 행동
└─ 같은 게임 수행

Step 3: 비교
├─ 인간 vs LLM의 협력률
├─ 구체적 선택 (배반 vs 협력)
└─ 신뢰도 평가
```

### 📊 실험 & 결과

**주요 발견 1**: **협력 패턴 복제**
- Public Goods Game:
  - 인간 평균 기여: 60%
  - LLM 평균 기여: **58%** (거의 동일!)
  
**발견 2**: **개인차 포착**
- 어떤 사람은 항상 협력 → LLM도
- 어떤 사람은 항상 배반 → LLM도
- **상관도: r = 0.82** (높음)

**발견 3**: **합리성의 인간적 편차**
- 이론적 Nash 균형: 모두 배반
- 실제 인간: 40% 협력
- **LLM: 43% 협력** (인간 수준)

**발견 4**: **Llama의 우수성**
- GPT: 약간 편향 (배반 경향)
- Llama: 인간 행동 더 정확 복제

### ⚠️ 한계

1. **게임 수준**: 단순 게임만 (복잡한 상황?)
2. **시간**: 한 번의 라운드만 (장기 변화?)
3. **상황 압박**: "지금 당장 결정해"만 (준비 시간?)
4. **감정**: LLM이 협력할 때 "느낌"이 있나?
5. **정직성**: LLM이 기만할 수도 (Strategic Deception)

### 🚀 미해결 문제
- **다중 라운드**: 상대방 행동에 LLM이 반응?
- **신뢰 형성**: 반복 상호작용에서?
- **전략 학습**: 에이전트가 상대 전략 학습?
- **새로운 상황**: 게임 규칙 바뀌면?

### 💡 종식을 위한 통찰
✅ **성과**: LLM이 **인간의 협력 행동 82% 정확도로 복제**
✅ **의미**: 디지털 트윈으로서 신뢰도 높음
✅ **주의**: Llama > GPT (모델 선택 중요)
✅ **기회**: 협력 정책의 효과 사전 테스트

## 1️⃣2️⃣ People Are Highly Cooperative with Large Language Models, Especially When Communication Is Possible

### 📋 기본 정보
- **arXiv ID**: 2507.18639
- **저자**: Paweł Niszczota, Anna Iwanowska, Agnieszka Piotrowska
- **게시**: 2025년 7월
- **피인용수**: 72 | **조회수**: 1,700
- **기관**: 폴란드 심리학 & 경제학 연구

### 🎯 문제 정의
**핵심**: **인간은 LLM 에이전트와 협력할까?** 그리고 **다를까?**
- 기존: 인간-인간 협력 연구 (많음)
- 새로움: 인간-LLM 협력 비교
- 질문: "신뢰도 차이가 있는가?"

**왜 중요한가?**
- 미래: 인간-AI 협력 시대
- 정책: AI 에이전트를 신뢰할 수 있나?

### 📚 기존 연구
- **인간 협력**: 심리학, 경제학 (풍부한 데이터)
- **인간-AI**: 새로운 분야
- **신뢰**: 얼굴, 이름, 성별 영향 연구 (인간만)
- **한계**: 인간-LLM 정량적 비교 부재

### 🔧 저자의 접근 방법

**실험 설계**:
```
조건 1: 인간 vs 인간 (죄수의 딜레마)
조건 2: 인간 vs LLM 에이전트
조건 3: 인간 vs LLM (통신 가능)
조건 4: 인간 vs LLM (이전 인간과의 상호작용 있음)

측정:
- 협력율 (%)
- 신뢰도 평가
- 커뮤니케이션 효과
```

### 📊 실험 & 결과

**발견 1**: **협력율의 차이**
```
인간 vs 인간: 65% 협력
인간 vs LLM: 55% 협력 (10% 낮음)

해석: 인간이 LLM을 약간 덜 신뢰
```

**발견 2**: **통신의 강력한 효과**
```
통신 없음: 55% (위와 같음)
통신 있음: 88% (33% 증가!)

해석: "대화할 수 있으면" AI도 인간처럼 신뢰
```

**발견 3**: **스필오버 효과**
```
먼저 인간과 협력 (높은 신뢰)
→ 그 다음 LLM과도 협력 증가 (학습 전이)

해석: 앞의 긍정 경험이 LLM 신뢰에 영향
```

**발견 4**: **개인차**
- 일부 사람: AI를 완전히 신뢰
- 일부 사람: AI를 완전히 불신
- 상관도: 약함 (개인 성향의 다양성)

### ⚠️ 한계

1. **게임 단순성**: 한 번의 선택만 (장기 신뢰?)
2. **LLM 투명성**: "너는 AI야"라고 명시 (현실은?)
3. **문화 한계**: 폴란드 표본만 (다른 문화?)
4. **시간**: 한 세션만 (반복 노출은?)
5. **비용**: 모든 LLM이 아니라 특정 모델만

### 🚀 미해결 문제
- **속임 감지**: 사람이 LLM의 기만 알아채는가?
- **신뢰 붕괴**: 한 번의 배신 후 복구?
- **장기 협력**: 몇 개월 반복하면?
- **혼합**: 인간 팀 vs AI 팀?

### 💡 종식을 위한 통찰
✅ **현상**: 인간이 AI 에이전트를 **약간 덜 신뢰** (10% 차이)
✅ **기회**: **통신 가능하면 신뢰 88%** (거의 인간 수준)
✅ **메시지**: 투명한 소통이 신뢰 구축의 핵심
✅ **응용**: 사회 시뮬레이션에서 신뢰도 모델링 필수
